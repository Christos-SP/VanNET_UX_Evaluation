# Αξιολόγηση εμπειρίας χρήσης (UX) της εφαρμογής VanNET
Σκοπός της σελίδας αυτής είναι να παρέχει το απαραίτητο υλικό για την αξιολόγηση της εμπειρίας χρήστη (User Experience) της εφαρμογής VanNET. Σε περίπτωση που αποφασίσετε να αξιολογήσετε την εφαρμογή, παρακαλούμε να κατεβάσετε στον υπολογιστή ή/και στο κινητό σας το παρακάτω υλικό.  

Το υλικό που προσφέρεται καλύπτει την αξιολόγηση της εμπειρίας χρήστη με βάση την: 
1) μέθοδο του ερωτηματολογίου (questionnaire) και
2) μέθοδο καταγραφής της δραστηριότητας του χρήστη (activity logging)

Η ομάδα μας επέλεξε για την αξιολόγηση εμπειρίας χρήστη να χρησιμοποιήσει την ελληνική έκδοση του [ερωτηματολογίου UEQ](https://spiros1979.github.io/Vannet/index.html), ενώ ως μέθοδος καταγραφής της δραστηριότητας του χρήστη (activity logging) χρησιμοποιείται το freeware [πρόγραμμα mousotron](https://spiros1979.github.io/Vannet/index.html).

Η αξιολόγηση της εμπειρίας χρήστη βασίζεται σε συγκεκριμένα σενάρια χρήσης (use scenarios), τα οποία ο χρήστης/αξιολογητής της εφαρμογής VanNET καλείται να διεκπεραιώσει.  
| Σενάρια Χρήσης | |  Concept Σεναρίου                  |  
| :-----------: | :-----------: | :-----------: |    
| Σενάριο Χρήσης 1 | [Link](https://spiros1979.github.io/Vannet/index.html) |  Ο χρήστης ελέγχει το επίπεδο νερού στις δεξαμενές αποθήκευσης, καθώς και το ιστορικό κατανάλωσης νερού στις δεξαμενές, ώστε να είναι προετοιμασμένος για τον επερχόμενο καύσωνα.   |    
| Σενάριο Χρήσης 2 |[Link](https://spiros1979.github.io/Vannet/index.html)|  Ο χρήστης καλείται να ελέγξει την πρόγνωση του καιρού, μέσω της web εφαρμογής και μέσω της εφαρμογής στο κινητό του, ώστε, ανάλογα τις καιρικές συνθήκες, να προγραμματίσει πότισμα του κήπου.  |
