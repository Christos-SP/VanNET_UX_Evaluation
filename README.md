# Αξιολόγηση εμπειρίας χρήσης (UX) της εφαρμογής VanNET
Σκοπός της σελίδας αυτής είναι να παρέχει το απαραίτητο υλικό για την αξιολόγηση της εμπειρίας χρήστη (User Experience) της εφαρμογής [VanNET](https://github.com/spiros1979/Vannet/tree/Arduino-Uno-R4-Wifi). Σε περίπτωση που αποφασίσετε να αξιολογήσετε την εφαρμογή, παρακαλούμε να κατεβάσετε στον υπολογιστή ή/και στο κινητό σας το παρακάτω υλικό.  

Το υλικό που προσφέρεται καλύπτει την αξιολόγηση της εμπειρίας χρήστη με βάση την: 
1) μέθοδο του ερωτηματολογίου (questionnaire) και
2) μέθοδο καταγραφής της δραστηριότητας του χρήστη (activity logging)

Η ομάδα μας επέλεξε για την αξιολόγηση εμπειρίας χρήστη να χρησιμοποιήσει την ελληνική έκδοση του [ερωτηματολογίου UEQ](https://ueq-online.org/), ενώ ως μέθοδος καταγραφής της δραστηριότητας του χρήστη (activity logging) χρησιμοποιείται το freeware [πρόγραμμα mousotron](https://www.blacksunsoftware.com/mousotron.html). 

Οι αναλυτικές οδηγίες που περιγράφουν όλη την διαδικασία της αξιολόγησης βρίσκονται [εδώ](https://github.com/Christos-SP/VanNET_UX_Evaluation/blob/main/Inventory/%CE%9F%CE%B4%CE%B7%CE%B3%CE%AF%CE%B5%CF%82.pdf).

Για την αξιολόγηση της ευχρηστίας της εφαρμογής VanNET που καλείστε να φέρετε εις πέρας, είναι απαραίτητη η καταγραφή μιας σειράς παραμέτρων όπως:  

•	ο αριθμός των clicks που κάνετε προσπαθώντας να ενεργοποιήσετε την εργασία που σας ζητήθηκε στο αντίστοιχο σενάριο χρήσης,   
•	ο συνολικός αριθμός των πλήκτρων που πιέσατε, κατά την διάρκεια χρήσης της εφαρμογής VanNET,  
•	ο συνολικός χρόνος που χρειαστήκατε για την ενεργοποίηση κάποιας λειτουργίας που σας ζητήθηκε στο αντίστοιχο σενάριο χρήσης  


Οι παράμετροι αυτοί, θα μας βοηθήσουν να αξιολογήσουμε πόσο εύχρηστη είναι η εφαρμογή VanNET και σε επόμενο στάδιο, να βελτιώσουμε την ευχρηστία της, μέσω στοχευμένων παρεμβάσεων. Για την καταγραφή των παραπάνω παραμέτρων με αντικειμενικό και αμερόληπτο τρόπο, χρησιμοποιούμε την δωρεάν εφαρμογή Mousotron, την οποία σας ζητούμε να κατεβάσετε από το [Link](https://www.blacksunsoftware.com/downloads/Mousotronsetup.exe) και να εγκαταστήσετε στον υπολογιστή σας. 

Μετά από την διεκπεραίωση κάθε σεναρίου χρήστη, καλείστε να καταχωρήσετε τις παραμέτρους από το πρόγραμμα mousotron στην φόρμα εισαγωγής στοιχείων που θα βρείτε [εδώ](https://github.com/Christos-SP/VanNET_UX_Evaluation/blob/main/Inventory/Measures_Form.pdf), καθώς επίσης και το [ερωτηματολόγιο UEQ](https://github.com/Christos-SP/VanNET_UX_Evaluation/blob/main/Inventory/UEQ_Greek.pdf) για την αξιολόγηση εμπειρίας χρήστη.

Επιπλέον, για λόγους στατιστικής και μόνο, ζητείται από τον χρήστη να συμπληρώσει τα δημογραφικά του στοιχεία, ώστε να σκιαγραφηθεί το προφίλ του. Είναι σημαντικό να τονιστεί ότι δεν απαιτείται κανένα προσωπικό στοιχείο, όπως ονοματεπώνυμο, διεύθυνση κατοικίας, αριθμό τηλεφώνου κτλ.
